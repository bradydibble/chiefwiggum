### Added

- Fix plan writer utility
- Cost tracking and token estimation scripts
- Response analyzer script

- TUI unsplit() for zoomed views (TASKS/INSTANCES only)
- Display bug with overlay panels

### Added
- Debug logging for TUI operations

### Fixed

### Added
- Initial TUI dashboard with Rich library
- Task claiming and coordination system
- Ralph instance lifecycle management
- SQLite-based coordination database

### Changed
